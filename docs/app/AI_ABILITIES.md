# AI機能・能力詳細 (AI Abilities)

本ドキュメントでは、現在実装されているアプリケーション内で、AI（エージェント）が自律的またはユーザーの指示に応じて実行できる機能（ツール）について説明します。

## 1. コンテキストを維持した自然言語対話
- **文脈の理解と記憶**: `@google/adk`のメモリ機能を活用し、セッション内の過去の会話履歴を踏まえた自然な対話が可能。
- **ペルソナの動的変更**: 選択されたエージェントのシステムプロンプトに従い、翻訳アシスタントや専門家など、指定された役割・トーンで応答する能力。

## 2. ウェブ検索 (Tavily Search)
- **最新情報の取得**: Tavily APIを利用してウェブ検索を自律的に実行し、最新のニュース、イベント、またはAIの学習データに含まれない外部知識を取得して回答に組み込む能力。（※Tavily APIキーの設定が必要）

## 3. ユーザーメモの連携・操作 (User Memos Integration)
AIはサイドバーにある「ユーザーメモ」に対して、ツールを通じて直接アクセス・操作することができます。

- **メモ一覧の取得**: 現在保存されているメモのタイトル一覧を取得し、どのような情報が存在するかを把握する機能。※アクセス対象は、AIエージェント自身（エージェント名）またはユーザーの現在アクティブなペルソナが公開対象（ターゲットオーディエンス）に含まれるメモのみに制限されます。
- **メモの読み取り**: 特定のメモのタイトルを指定して、その内容（本文）を読み取り、ユーザーの質問に対するコンテキストとして活用する機能。※アクセス制限があり、自身またはユーザーのアクティブペルソナが公開対象に設定されていないメモは読み込めません。
- **メモの作成・更新**: 対話を通じて得た重要な情報やまとめなどを、AI自らが新しいメモとして作成、あるいは既存のメモを更新して保存する機能。新規作成・更新時には、自動的にユーザーのアクティブペルソナ名とAIエージェント名がターゲットオーディエンス（公開対象）に紐付けられます。
